# Detection of Lung Cancer Mutation Using Adaptive Boosting

This project uses **Adaptive Boosting (AdaBoost)** to detect **EGFR lung cancer mutations** based on clinical and morphological features.

## Dataset
- 80 lung cancer patients  
- Clinical and morphological features  
- Preprocessing includes:
  - Missing value handling  
  - Outlier removal  
  - ANOVA feature selection  

## Method
- Model: **AdaBoost**
- Hyperparameter tuning (learning rate & estimators)

  - Cross-validation

## Results
- **Accuracy:** ~83%  
- **AUC:** ~0.90
